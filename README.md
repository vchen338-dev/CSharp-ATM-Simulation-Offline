# ATM Simulation System

This is a testing and educational project that simulates the functionalities of an ATM system. It is designed for learning purposes and does not represent any real banking system.

![ATM Simulation System](https://img.shields.io/badge/status-unfinished-red)
![License](https://img.shields.io/badge/license-MIT-blue)

## Upcoming Features

- Account Registration UI and Function
- Deposit Function
- Card Function
- Account Setting Function
- Withdraw Function
- Transfer Function
- Microsoft Excel DataBase for Offline or SQLite

## Disclaimer

“This ATM Simulation System is a testing and educational project only. It does not represent any real banking system or financial institution. All names, accounts, and transactions used within this application are fictional and created for demonstration purposes. No real money is involved or will ever be processed. By using this system, you acknowledge that all displayed data and results are for practice, simulation, and learning purposes only.”
